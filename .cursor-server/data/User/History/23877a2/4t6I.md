## Quick Start

### 1. Install Dependencies

```bash
pip3 install --break-system-packages requests aiohttp Brotli
```

### 2. Configure Wallets

Edit `main_fast.py` and add your wallet addresses:

```python
WALLETS: list[str] = [
    "0xYourWalletAddress1",
    "0xYourWalletAddress2",
    "0xYourWalletAddress3",
    # Add more wallets...
]
```

### 3. Run the Bot

**Standard version:**
```bash
python3 main.py --offset 0
```

**Ultra-fast version (Recommended):**
```bash
python3 main_fast.py --offset 0
```

**As systemd service:**
```bash
# Create service file
sudo nano /etc/systemd/system/christmas-bot.service

# Paste this content:
[Unit]
Description=Christmas Claim Bot
After=network-online.target

[Service]
Type=simple
User=root
WorkingDirectory=/path/to/itsbot
ExecStart=/usr/bin/python3 -u /path/to/itsbot/main_fast.py --offset 0
Restart=always
RestartSec=10
StandardOutput=append:/path/to/itsbot/output.log
StandardError=append:/path/to/itsbot/error.log

[Install]
WantedBy=multi-user.target

# Enable and start
sudo systemctl daemon-reload
sudo systemctl enable christmas-bot.service
sudo systemctl start christmas-bot.service
```

## How It Works

### Timing

1. **XX:59:50-XX:59:57** - Warms up connections
2. **XX:00:00-XX:00:08** - Burst attack window (claims hour that just unlocked)
3. Fires 7-12 concurrent requests with randomized timing
4. Logs all attempts to `advent_log_fast_0.txt`

### Wallet Rotation

With 3 wallets and offset 0:
- Hour 1, 4, 7, 10, 13, 16, 19, 22 ‚Üí Wallet #1
- Hour 2, 5, 8, 11, 14, 17, 20, 23 ‚Üí Wallet #2
- Hour 3, 6, 9, 12, 15, 18, 21, 24 ‚Üí Wallet #3

Change offset to shift which wallet is used for each hour.
